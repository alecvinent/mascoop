OHRMS Service Request

Manages the Technical requirements of Employees

Cybrosys Technologies

Overview

This module is an application for handling the Service Request from employees.After an employee submits his service request to the administrator he will assign a employee to perform the service as 'Service Executer'. After the executer finish the service he change the state as 'Checked',then the requested employee will approve reject the service.

Features

  • ★   Allows employees to raise a request and view its status.
  • ★   Service Type and Item For Service is defined.
  • ★   Assigning executors and approvers at multiple levels.

STEPS

Select the User and set the "Inventory" field to 'Manager'
Select the User and set the "Service" field to 'Service Executer'
Fill the mandatory fields and click "Submit" to send request
Choose the Executer for the request and click on 'Assign'
Click on the 'Checked' button after solving the request
Click 'Approve' if the request is satisfied
Now the Service is approved
