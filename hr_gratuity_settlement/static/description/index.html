Overview

Employee Gratuity Settlement is a component of Open HRMS suit. This module manages employee settlements during resignation process. A Gratuity Settlement can create only if the Resignation Request of the employee is approved. The Gartuity amount is calculated by the basic equation, Last drawn salary (basic salary plus dearness allowance) * number of completed years of service * 15/26.
